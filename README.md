# MusicMetrics

MusicMetrics is a static-first, global English-language music analytics MVP built with Next.js 14, TypeScript, and Tailwind CSS.

## What is included

- Homepage with hero, featured charts, artists, tracks, countries, and insights
- Detail pages for charts, artists, tracks, countries, and insights
- Local typed mock data to simulate a launch-ready analytics site
- SEO essentials: metadata, sitemap, robots, manifest
- Netlify-ready configuration

## Run locally

```bash
npm install
npm run dev
```

## Deploy on Netlify

1. Push these files to your `musicmetrics-net` GitHub repository.
2. Connect the repo to Netlify.
3. Build command: `npm run build`
4. Publish directory: leave default for Next.js plugin.
5. Deploy.
